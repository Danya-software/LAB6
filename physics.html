Літак ІЛ-62 має чотири двигуни, сила тяги кожного 103 кН. Яка корисна потужність двигунів при польоті літака зі швидкістю 864 км/год?

Дано:

v = 864 км/год = 240 м/с
F = 1,03 × 105 Н

Знайти:

N - ?

Рішення:

Корисна потужність N двигунів дорівнює відношенню механічної роботи A до часу t:

N = A/t

Механічна робота дорівнює:

A = Fs

Отже, можемо записати:

N = A/t = Fs/t

Оскільки v = s/t, ми маємо:

N = Fv

Підставляючи значення, отримаємо:

N = 240 м/с * 1,03 × 105 Н ≈ 2,5*107 Вт = 250 кВт.

Відповідь: N = 250 кВт
